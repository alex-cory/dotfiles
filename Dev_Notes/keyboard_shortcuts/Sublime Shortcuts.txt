Currently Learning ----------------------------------------------


⌘ ⇧ V 		: Paste paste PREVIOUS clipboard items

^ D 		: Highlight everything within current tag
  or
⌘ ⇧ J
  or
⌘ ⇧ A 		: Select everything within tags
  or
⌘ ⇧ __ 		: Select Scope (meaning if it's a paragraph, select the whole paragraph)

^ ⇧ M 		: Select Brackets

⌘ ⌥ . 		: Complete/close tag

⌘ ⌥ G 		: find next occurrence of current word

Editing ----------------------------------------------------------
⌘ ⌥ ⇧ ◄ ► ▲ ▼ ⏎

⌘ KK 		: Delete from cursor to end of line

⌘ ⇧ D 		: Duplicate line/selection

^ ⌥ ▲/▼ 	: Move line up/down

⌘ Enter 	: Insert lines after

⌘ ⇧ Enter 	: Insert lines before

⌘ KU 		: Uppercase

⌘ KL 		: Lowercase

⌘ / 		: Comment entire line

⌥ ⇧ / 		: Comment portion of the line that is selected



Selecting ----------------------------------------------------------
⌘ ⌥ ⇧ ◄ ► ▲ ▼ ⏎

⌘ ⇧ __ 		: Select Scope (meaning if it's a paragraph, select the whole paragraph)

⌘ D 		: Highlight the word the courser is closest to 

⌘ L 		: Highlight this row and move down a line

^ ⇧ M 		: Select Brackets

⌘ ⌥ ▲/▼ 	: Move Screen Up/Down 1 line





Mulit - Selecting ----------------------------------------------------------
⌘ ⌥ ⇧ ◄ ► ▲ ▼ ⏎

⌘ D 		: Select next occurrence

⌘ K 		: Skip occurrence

⌘ U 		: Undo Selection

^ ⌘ G 		: Select All occurrences

^ ⇧ ▲/▼		: Add previous/next line courser

^ ⇧ M 		: Select all inside parenthesis

⌘ ⇧ J 		: Expand Selection until indentation

^ M 		: Go to closing/opening parenthesis



Search ----------------------------------------------------------
⌘ ⌥ ⇧ ◄ ► ▲ ▼ ⏎

⌘ R 	: Show all functions/Methods

⌘ P 	: Go To Anything (within the scope of your window)
