


Currently Learning ----------------------------------------------

⌘ ⇧ ] 	: Next Tab

⌘ ⇧ [ 	: Previous Tab





Most Important --------------------------------------------------
⌘ ⌥ ⇧ ◄ ► ▲ ▼

⌘ D 	: split pane left-right

⌘ ⇧ D 	: split pane top-bottom 

⌘ ⇧ ] 	: Next Tab

⌘ ⇧ [ 	: Previous Tab

⌘ ⇧ H   : Command History

Tab navigation ------------------------------------------------------
⌘ ⌥ ⇧ ◄ ► ▲ ▼

⌘ T 	: Open New Tab

⌘ ⇧ ] 	: Next Tab

⌘ ⇧ [ 	: Previous Tab






Pane navigation ----------------------------------------------------------
⌘ ⌥ ⇧ ◄ ► ▲ ▼

⌘ D 	: split pane left-right

⌘ ⇧ D 	: split pane top-bottom 

⌘ ] 	: Next Pane

⌘ [ 	: Previous Pane

⌘ K 	: Clear Buffer (clear)

⌘ L 	: clear lines (Bash command)

⌘ + 	: make font larger

⌘ - 	: make font smaller




Search ----------------------------------------------------------------
⌘ ⌥ ⇧ ◄ ► ▲ ▼

⌘ F 	: Open Search Bar

⌘ G 	: Find Next



Random ------------------------------------------------------------
⌘ ⌥ ⇧ ◄ ► ▲ ▼

⌘ ⌥ I 	: input to all panes in current tab
