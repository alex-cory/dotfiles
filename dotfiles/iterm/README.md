# iTerm Notes


## iTerm (move around quickly)

#### Quickly Jump to Beginning and End of Lines
set in iTerm > Preferences > Keys > Global Shortcut Keys
    FOR ACTION               SEND
    ⌘←  "Send HEX CODE"      0x01
    ⌘→  "Send HEX CODE"      0x05

#### Quickly Jump Through Words
set in iTerm > Preferences > Profiles (click profile) > Keys
    FOR  ACTION         SEND
    ⌥←  "SEND ESC SEQ"  b
    ⌥→  "SEND ESC SEQ"  f
