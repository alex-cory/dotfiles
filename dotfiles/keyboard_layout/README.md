Keyboard Layout
===============

To use this you need the [Karabiner](https://pqrs.org/osx/karabiner/) mac application.


Keyboard Shortcuts
------------------

##### Space + A
`open Slack`

##### Space + S
`toggle iTerm2`

##### Space + D
`open Google Chrome`

##### Space + J
`⌘ + shift + left`

##### Space + L
`⌘ + shift + right`

##### Space + R
`current application -> browser -> refresh page`

Super Keys
----------

##### Hold Tab
`⌘ + shift`

##### Hold Space
`⌘ + ⌥ + ctrl + shift`

##### Hold Single Quote
`ctrl`

##### Hold Caps Lock
`ctrl`

##### Tap Caps Lock
`escape`
